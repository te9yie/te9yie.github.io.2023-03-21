GitHub Actionsのcheckoutアクション

https://github.com/actions/checkout

Gitでファイルごとの更新日付を取得するようにしていたが、GitHub Actionsを使ったらうまく取得できていなくて全て同じ日時になっていた。 調べてみるとactions/checkout@v2はデフォルトで最新履歴だけを取得するようになっていた。fetch-depthを0に設定して対応した。
