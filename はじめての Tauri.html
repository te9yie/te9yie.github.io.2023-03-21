はじめての Tauri

Tauri はバックエンドが Rust の Electron みたいなものみたい。

https://tauri.app/

Electron はフロントエンドが Chrome、バックエンドが Node.js。Tauri はフロントエンドが OS ネイティブの Web レンダラで、バックエンドが Rust ということらしい。

TauriElectron
フロントエンドOS ネイティブの Web レンダラChrome
バックエンドRustNode.js

公式サイトのガイドを参考にとりあえず動くものを作ってみる。
