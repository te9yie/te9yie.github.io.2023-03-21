Git でファイルの更新日時を取得する

git log -1 --pretty="%ad" -- <file>

日時のフォーマットを指定することもできる。

git log -1 --pretty="%ad" --date=format:"%Y-%m-%d %H:%M:%S" -- <file>

ref Git - コミット履歴の閲覧

ファイルの更新日時として Git でコミットした時間が使えたらいいなと思った。
